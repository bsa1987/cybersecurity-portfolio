## Overview
This project simulates a real-world security audit for **Botium Toys**, a fictional e-commerce toy company.  
It includes a full security audit report and a compliance checklist aligned with industry best practices.

## Files Included
- **Botium_Toys_Full_Audit_Report.pdf**  
  Complete security audit with findings, risks, and recommendations.

- **Botium_Toys_Audit_Report_With_Checklist.pdf**  
  Audit checklist covering policies, processes, standards, and security controls.

## Skills Demonstrated
- Governance, Risk & Compliance (GRC)
- Security auditing methodologies
- Identification of security gaps
- Risk analysis and prioritization
- Documentation and reporting
- Compliance assessment

## Project Purpose
To practice performing a structured cybersecurity audit and documenting findings as part of a growing GRC and cybersecurity portfolio.
